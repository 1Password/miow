# wio

[![Build status](https://ci.appveyor.com/api/projects/status/tc5lsxokjk86949l?svg=true)](https://ci.appveyor.com/project/alexcrichton/wio)

A zero overhead Windows I/O library focusing on IOCP and other async I/O
features.

```toml
# Cargo.toml
[dependencies]
wio = "0.1"
```
